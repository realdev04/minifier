# Minifier

A Simple yet Powerful and Ultrafast File Minifier.<br>
Minifier will Minify your Code within Milliseconds.

## Usage

`minifier [Input File] [Output File]` <br>
Ex:<br>

```shell
minifier styles.css styles.min.css
```
